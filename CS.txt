CS -> PEP 8


Étape 1: Installation de flake8 et black

pip install flake8 black

Étape 2: Configuration de flake8

Créez un fichier de configuration ".flake8" ->

[flake8]
max-line-length = 120
exclude = .git,__pycache__,venv
extend-ignore = E203, W503

Étape 3: Configuration de black

Créez un fichier de configuration "pyproject.toml" ->

[tool.black]
line-length = 120
exclude = '\.git|__pycache__|venv'



Étape 4: Utilisation de flake8 et black


# Linting avec flake8
flake8 .

# Formatage avec black
black .
